# AlphaRoute - 智能量化跟单平台

AlphaRoute 是一款专为加密货币投资者打造的国际化智能量化跟单平台。平台采用先进的订阅制商业模式，通过自动化执行顶级交易员的策略信号，帮助用户在 Binance、OKX、Bybit、Bitget 和 Gate.io 等主流交易所实现资产的稳健增值。

## 核心商业模式：订阅制

AlphaRoute 摒弃了传统的利润分成模式，采用更加透明、公平的**订阅制**。用户只需支付固定的订阅费用，即可享受全自动的跟单服务，所有交易利润 100% 归用户所有。

### 订阅档位
平台提供两种订阅档位，满足不同资金体量用户的需求（具体价格由管理员在后台配置）：
- **基础档 (Basic)**：适合小额账户体验和日常交易。
- **进阶档 (Advanced)**：适合大额账户，享受更高的资金容量和专属服务。

### 支付方式
- **去中心化支付**：用户直接使用 USDT (支持 TRC20 和 BSC 网络) 支付到平台为每个订单生成的专属独立地址。
- **自动扫链确认**：平台内置自动扫链引擎，用户转账后系统自动确认并开通订阅权限，全程无需人工干预。
- **无资金池风险**：平台不设“充值”和“提现”功能，用户的交易资金始终安全地存放在自己的交易所账户中，平台仅通过 API 密钥获取交易权限。

### 推广奖励机制
- **7天免费试用**：所有新注册用户均可获得 7 天的免费试用期，零成本体验平台功能。
- **邀请奖励**：用户邀请好友订阅，可获得被邀请人订阅时长一定比例的奖励天数。被邀请人后续续费，邀请人可持续获得奖励天数。

## 平台功能模块

### 用户端 (User Portal)
- **仪表盘 (Dashboard)**：直观展示用户的累计盈利、累计亏损、净盈亏、跟单总数及当前持仓数量。
- **策略中心 (Strategies)**：展示平台提供的优质交易策略，用户可一键订阅并配置跟单倍数。
- **订单记录 (Orders)**：实时同步交易所的开仓、平仓记录，清晰展示每笔交易的已实现盈亏和净盈亏（扣除手续费后）。
- **我的订阅 (Subscription)**：管理当前的订阅状态、剩余天数，并提供便捷的续费入口。
- **邀请奖励 (Invite Reward)**：获取专属邀请链接，查看团队成员列表、团队累计消费以及自己获得的奖励天数。
- **API 绑定 (API Keys)**：安全绑定各大交易所的 API 密钥（仅需交易权限，严禁开启提现权限）。

### 管理端 (Admin Portal)
- **数据大盘 (Dashboard)**：全局掌控平台运营数据，包括注册用户数、累计盈利/亏损、异常订单数，以及平台的订阅收入统计。
- **订阅管理 (Subscription)**：
  - **钱包配置**：配置平台的 HD 钱包助记词，以及 TRC20 和 BSC 网络的资金归集主地址。
  - **价格配置**：灵活设置基础档和进阶档的月度、季度、年度订阅价格。
  - **订单审核**：查看所有用户的订阅支付记录，系统自动扫链确认，也支持管理员手动审核异常订单。
- **用户管理 (Users)**：查看所有用户的详细信息、订阅天数、消费记录及跟单历史，支持一键启停用户权限。
- **信号源管理 (Signals)**：配置和管理对接的外部交易策略信号源。
- **订单监控 (Orders)**：全局监控所有用户的跟单执行情况，及时发现并处理异常订单。

## 技术架构与安全性

- **前端框架**：React + Vite + TailwindCSS + shadcn/ui，提供流畅、精美的国际化用户体验。
- **后端服务**：Node.js + Express + tRPC，确保前后端类型安全和高效通信。
- **数据库**：MySQL (通过 Drizzle ORM 管理)，保证数据的强一致性和高可靠性。
- **跟单引擎**：独立运行的 Copy Engine，通过 WebSocket 实时监听信号源，并利用各交易所的 REST API 毫秒级并发执行跟单指令。
- **安全保障**：
  - API 密钥采用 AES-256-GCM 工业级加密存储。
  - 平台不触碰用户交易本金，彻底杜绝资金池跑路风险。
  - 独立的订阅支付地址和自动归集机制，保障平台收入安全。

## 部署与运行

平台采用 PM2 进行进程管理，确保服务的高可用性：
- `copy-trading`：主服务进程，包含 Web API、跟单引擎和自动扫链任务。
